Git拉取非master分支的解决方案

在拉取项目时，默认是master分支，要想切换分支到非master上，就下面的操作做简单的解释，想在本地创建一个与远程想要拉取分支一样的名称(ps：便于区分)，将本地切换到该新建分支上，最后设置拉取流为远程分支和本地新建分支上，最后pull下来该分支上的内容，大功告成~~ 补充：git branch -a可以查看本地和远程所有分支以及显示目前本地处于哪一个分支上。

git branch Develop
git checkout Develop
git branch --set-upstream-to=origin/Develop Develop
git pull
