centos更新jdk版本
Table of Contents

    对于通过yum安装jdk的centos系统，我们可以采用下面的方式升级jdk版本：

    ## 查看当前的jdk版本
sudo yum list java-1.7.0-openjdk

## 卸载原来jdk1.7
sudo yum remove java-1.7.0-openjdk.x86_64

## 安装jdk1.8
sudo yum install java-1.8.0-openjdk.x86_64

    这种方式是通过yum直接安装jdk，比较简洁方便。

    对于不是通过这种方式安装jdk的主机，我们只能将之前的安装文件找出来并删除，再下载新的jdk软件包进行解压，最后修改全局JAVA变量值，这样才能完成JDK的升级，具体的操作流程有待更新。
