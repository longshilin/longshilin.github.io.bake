在DOS命令行下执行命令运行Java文件：
1、运行jar包 java -jar test.jar
2、编译Java文件 javac test.java
3、运行Java的字节码文件 java test
4、调试Java文件 jdb test.java
5、生成Java doc文件 javadoc test.java
